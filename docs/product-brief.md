# Product Brief

Apple Orchestrator AI Local is a Mac-first local workflow orchestration app.

The Mac runs Hermes, Ollama, workflow skills, workflow JSON, matter workspaces, and the local observability database. iPhone and iPad are controllers for starting runs, reviewing status, approving human-in-the-loop tasks, and viewing outputs.

## Core Promise

Local means local:

- local Hermes
- local workflow files
- local skills
- local run state
- local matter workspaces
- local model execution through Ollama

External systems are accessed only when the user configures MCP tools/connectors for them.

## First Audience

The first audience is legal students and young legal professionals. The initial workflow pack should be based on prior OrchestratorAI Local legal work.

The same architecture should later support accountants and other professions through domain workflow packs.
