# MCP Resources

External files, databases, APIs, and systems are reached through MCP-backed resources and connectors.

Hermes skills can request resources such as:

- local folder
- iCloud Drive folder
- SQL Server
- SQLite
- CSV/JSON file
- SharePoint
- OneDrive
- iManage
- NetDocuments
- Clio
- REST API

If a required resource is missing, Hermes asks the user to attach it. The local app records the resulting resource binding for observability and audit.
