# Legal Workflow Pack

The first workflow pack should target legal students and young legal professionals.

Candidate workflows:

- matter intake
- document summary
- chronology builder
- fact extraction
- issue spotting
- discovery review
- memo drafting
- demand letter drafting
- motion outline

Each workflow should include:

- workflow JSON
- required skills
- sample test matter
- expected outputs
- validation checklist

The first pack should reuse and adapt prior OrchestratorAI Local legal work.
