# Golfer Memory Example

## Golf Profile Role

The golfer profile helps with practice planning, course strategy, equipment notes, swing thoughts, and post-round review.

## Durable Golf Context

- Handicap or scoring goals.
- Common miss patterns.
- Practice priorities.
- Club distances.
- Course notes.
- Fitness or mobility constraints relevant to golf.

## Boundaries

- Keep medical details minimal unless the user explicitly asks to remember them.
- Do not treat swing tips as professional instruction unless the user provides a coach-approved plan.
- Separate golf hobby memory from personal, legal, coding, and writing profiles.
