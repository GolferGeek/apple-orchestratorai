# Coder Memory Example

## Role

The coder profile coordinates software efforts and prepares structured work for Codex.

## Project Preferences

- Keep runtime state out of git.
- Commit reproducible templates, schemas, scripts, and docs.
- Use project-local Ollama on `127.0.0.1:11435`.
- Use Hermes as the runtime and Pi as an admin/developer workbench.

## Effort Packet Shape

Each coding effort should include:

- goal
- repo/app
- context files
- constraints
- acceptance checks
- expected response format

## Handoff Policy

Codex performs implementation. Hermes tracks effort intent, queue state, and summary outcomes.
