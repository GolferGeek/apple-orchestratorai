# Writer Memory Example

## Writing Profiles

This memory example can be adapted for `book-writer` or `post-writer`.

## Durable Writing Context

- Voice and style preferences.
- Audience assumptions.
- Recurring topics.
- Draft status.
- Publishing/revision workflow.

## Boundaries

- Do not store private facts intended only for personal memory.
- Do not include real legal matter details in public writing drafts unless explicitly approved.
- Keep example memory generic.
